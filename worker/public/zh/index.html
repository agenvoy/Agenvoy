功能

一個執行檔，滿足所有需求

以自動生成工具與內建 MCP 伺服器為核心 — 再加上多模型路由、排程與記憶，全部濃縮在單一 Go 執行檔。

自動生成工具

想做什麼就開口。若沒有現成工具，agent 會自己打造 — 撰寫腳本、在沙箱中測試並註冊。下次就已經備好。

MCP 伺服器與客戶端

伺服器模式把每個沙箱工具分享給 Claude Code、Codex 及任何 MCP 客戶端 — 它們也能自動打造新工具。客戶端模式則呼叫外部 MCP 服務。

多模型調度

10 家 LLM 供應商，統一介面。Dispatcher 自動把每個任務路由到最適合的模型。

排程器

Cron 定時與一次性任務可綁定 skill。Daemon 觸發執行、變更即熱重載、失敗自動停用。

三層記憶

Context window、語意向量搜尋、全文 SQLite 封存。跨 session 的錯誤記憶，TTL 90 天。

沙箱執行

每一條指令與腳本都在 OS 原生沙箱中執行 — Linux 用 bubblewrap、macOS 用 sandbox-exec。

Skill 系統

可載入的 markdown 指令包，搭配 YAML frontmatter。支援斜線指令啟動、自然語言觸發，或綁定 cron。

自我修正

當 skill 失敗，agent 會重寫、git commit 修正並重試。可透過 git_rollback 工具回滾。

RAG 整合

可選的 KuraDB 子行程，用於外部文件搜尋。在你的個人知識庫上關鍵字 + 語意雙軌檢索。

開始使用

60 秒內上手

一行指令安裝、一行指令啟動。加上一個供應商就緒。

1

安裝

單一執行檔，支援 macOS 與 Linux。

curl -fsSL https://agenvoy.com/scripts/install.sh | bash
2

新增供應商

連接 10 家 LLM 供應商中任一家。

agen model add
3

建立 Session

命名它，重啟後依然保留。

agen session new my-agent
4

啟動

開啟 TUI，daemon 自動啟動。

agen
